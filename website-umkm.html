FAQ

Pertanyaan yang sering diajukan seputar paket UMKM

Apakah bisa pakai domain sendiri?
Bisa! Jika Anda sudah punya domain sendiri, kami bisa konfigurasi agar website menggunakan domain tersebut. Biaya tambahan hanya untuk setup awal (sekali bayar). Domain rijanara.web.id tetap bisa digunakan sebagai alternatif.

Bagaimana cara update produk?
Untuk Paket Basic, update produk dilakukan melalui admin panel sederhana atau hubungi kami via WhatsApp. Paket Bronze ke atas sudah termasuk dashboard admin yang lebih lengkap untuk kelola produk sendiri.

Apakah bisa upgrade paket?
Tentu bisa! Upgrade paket bisa dilakukan kapan saja. Data dan konten website Anda tetap aman, hanya fitur yang ditambahkan. Proses upgrade biasanya selesai dalam 1-2 hari kerja.

Berapa lama website bisa online?
Setelah pembayaran pertama dan data bisnis Anda lengkap, website biasanya online dalam 3-7 hari kerja. Untuk paket Custom, timeline disesuaikan dengan kompleksitas fitur yang diminta.
